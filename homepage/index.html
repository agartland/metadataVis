MetaVis: Interactive Exploratory Visualizations

An in-depth interactive visualization tool which integrates metadata to flexibly produce novel insight.

About MetaVis

MetaVis is a interactive visualization tool developed for internal Fred Hutch usage to enhance exploratory analysis and provide a wider and cleaner perspective on various problem spaces. The tool includes:

  • Direct data import through web interface
  • Interactive heatmap with flexible selection options
  • Responsive data table population upon heatmap selection
  • Interchangeable colorbar and metadata visualizations
  • Optional data filtering capabilities

Data Specifications

For usage, data provided must meet the following specifications:

  • 3 separate .csv files containing base data, row metadata, and column metadata respectively
  • Proper formatting for each file (correct number of rows/columns necessary)
  • For optimum performance, rows/columns should not exceed 300
